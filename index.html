Welkom, mijn naam is Jelle Swaelen en dit is mijn website. Deze website zal doorheen het jaar worden aangepast en al mijn GIP-taken kan je ook hier terugvinden.

Ik woon te Schilde, ben 17 jaar oud en volg uiteraard ITN op het Immaculata Instituut. Alles rond computers interesseert me en als ik me sportief wil bezig houden doe ik dit door te gaan fietsen of oefeningen te doen met gewichten.

Tot zo ver ik weet heeft technologie me altijd geïnteresseerd. Dit begon toen ik ongeveer 6 jaar oud was, toen kreeg ik mijn eerste spelconsole. Ik was zo overweldigd door wat ik er allemaal mee kon doen. Al die spelletjes die ik kon spelen met iets dat in mijn hand past. Mettertijd had ik al een hele collectie spelletjes en was ik meer te weten gekomen over andere spelconsoles.

Doorheen de jaren groeide mijn kennis, maar ook mijn interesse voor de wereld van technologie. Eén van de leukste dingen aan technologie is, dat alles constant verandert. Tot op de dag van vandaag leer ik nieuwe dingen bij over hardware, maar ook software en wat er zich bij wijze van spreken achter de schermen voordoet. Ik hoop later een job te vinden in de IT-sector of eventueel zelf een bedrijf te starten.

Technologie en alles errond heeft met nog nooit verveeld en ik heb er het volle vertrouwen in dat dit niet snel zal gebeuren.

Als er opmerkingen zijn over iets op mijn website mogen die altijd vermeld worden en ik wens je nog veel lees- en surfplezier.

C#

C# is de programmeertaal waar we het meeste over praten en oefeningen op maken in de klas. Zelf vind ik de taal redelijk interessant, ik snap ook waarom we over die taal leren en de oefeningen in deze taal zorgen ervoor dat iedereen een basis heeft in programmeren.


Javascript & jQuery

Javascript hebben we vooral vorig jaar gebruikt. Ook deze taal interesseert me, zeker het aspect dat je dingen in HTML met behulp van Javascript kan aanpassen. jQuery, een taal die een beetje lijkt op Javascript, vind ik ook zeer boeiend. De code die ik al heb gebruikt in jQuery doet altijd veel voor wat er staat. In vergelijking met Javascript vind ik jQuery soms handiger, omdat het minder code is dat hetzelfde doet.


HTML & CSS

Over HTML en CSS hebben we vorig jaar veel oefeningen gemaakt, nu houden we ons er ook mee bezig doordat iedereen aan zijn persoonlijke website is aan het werken. Wat ik persoonlijk leuk vind aan HTML en CSS is het visuele aspect bv. de mogelijkheid om elementen te veranderen afhankelijk van de grootte van het venster/ welk apparaat de bezoeker gebruikt. Met HTML en CSS houd ik me ook bezig in mijn vrijetijd, wat ik niet veel doe bij de andere talen.
